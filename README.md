# Excuses

Simple React + TypeScript app for generating excuses by category or building one from selected options.

## Requirements

- Node.js 20+

## Install

```bash
npm install
```

## Development server

```bash
npm start
```

The app runs with Vite (default: `http://localhost:5173`).

## Build

```bash
npm run build
```

Output is written to `dist/`.

## Run tests

```bash
npm test
```

## Preview production build

```bash
npm run serve
```
